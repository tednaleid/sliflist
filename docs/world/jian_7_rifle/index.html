Jian 7 Rifle Identifier

Jian 7 is the randomly-rolled successor to the New Monarchy Pulse of the same name from days gone. It’s a 390RPM Adaptive Pulse Rifle which is similar to the Last Perdition which also rolls in this slot.

  • The Jian’s base recoil is 80 which means that with a Counterbalance mod, it will be trend mostly vertical at 95. Contrast with the Last Perdition whose base Recoil is 63.
  • Perks are mostly similar with some slotting variations. There are no Rampage- based perks here because it doesn’t make it any easier to off enemies unless you’re at x 2 which is too situational for a solid recommendation.

Most everything written for the Last Perdition applies here as well.

👑 Make Way for the New Ruler (pvp)#

Firmly Planted and Zen Moment will help with Stability. If you haven’t given Firmly Planted a shot, it’s right at home on Pulse Rifles who have long(er) engagement ranges and are more amenable to the crouch. It’s a difference in playstyle that you have to work at, but you will absolutely be rewarded. This perk alone is why the Jian 7 and Last Perdition have become two of my most- used pulses.

Collector’s Edition Roll

  • Barrels: SLO-10 Post
  • Magazines: Ricochet Rounds, High-Caliber Rounds
  • Perks 1: Firmly Planted, Zen Moment
  • Perks 2: Rangefinder
  • Masterworks: Stability MW
